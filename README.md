# momentary-profiles-project
This is the repository of Quarto scripts, codebooks, figures, and tables for the Momentary Profiles project.<br />
The scripts, codebooks, figures, and tables are organized by subfolders.<br />
For example, the profile plot of participant X of the IPCS sample will be found in:<br />
* Figures folder<br />
-Profile Plots folder<br />
-IPCS Study<br />
-Participant X file
